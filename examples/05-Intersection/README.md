# Intersection

Let's try out a road intersection and communication between Traffic Lights!

## Hardware required

* Arduino Board
* Breadbord
* 6 LEDs
* 6 Resistors (220Ω-1000Ω should work)
* 2 Buttons (optional)
* Hook-up wires
