# Testing Callbacks

The library has various built-in event functions that allows dispatching functions that should be triggered by specific events. Let's test it by implementing custom callbacks.

## Hardware required

* Arduino Board
* 3 LEDs
* 3 Resistors (220Ω-1000Ω should work)
* Button (optional)
* Hook-up wires
